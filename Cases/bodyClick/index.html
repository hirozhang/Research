Verify the integrity of the download

To ensure the integrity of your downloaded software, the Installer generates an MD5 Checksum value for each platform-specific software installation file. You can use this value to verify that a downloaded file is intact. After you click on the software download link and launch the Installer page, you can see the MD5 Checksum value listed on the page. Each value listed is unique. After you finish downloading the software from the Installer page, you can calculate the MD5 Checksum value on your computer and compare it to the value on the Installer page. If the downloaded file is intact, the values will match.
